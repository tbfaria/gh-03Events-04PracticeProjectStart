# React Practice Project

Simple React app built with Vite.

## Project Setup

This project was created using Vite:

    npm create vite@latest
    cd project-name
    npm install

---

## Run the project

Install dependencies:

    npm install

Start the server:

    npm run dev

Open in browser:
http://localhost:5173

---

## Scripts

    npm run dev      # Start server
    npm run build    # Build project
    npm run preview  # Preview build

---

## Notes

- Use `npm install` locally
- Use `npm ci` in GitHub Actions
- Project is for practicing CI/CD
